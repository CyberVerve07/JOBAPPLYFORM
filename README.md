A simple and clean desktop-based Job Application Management System built using Java Swing.
The application allows users to submit job applications, view them in a dynamic table, update records, delete entries, and export data in CSV format.

This project is ideal for beginners learning Java GUI, OOP, and event-driven programming, while still looking professional enough for a portfolio.
